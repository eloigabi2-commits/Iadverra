# Google Ads — Setup de Alta Conversão | Iadverra

## Por que as campanhas atuais não convertem

O problema mais comum em Google Ads B2B com baixa conversão:
- Keywords muito amplas → cliques de quem não tem intenção de comprar
- Anúncio não pré-qualifica → chega gente que não é o cliente ideal
- Landing page pede demais antes de dar valor → abandono

---

## Estrutura da Campanha

### Tipo: Pesquisa (Search)
### Objetivo: Geração de leads (reuniões agendadas)
### URL de destino: `/diagnostico/`

---

## Keywords — Apenas Correspondência Exata e de Frase

Não use correspondência ampla. Só exata `[...]` e frase `"..."`.

### Grupo 1 — Automação de Processos
```
[automação de processos empresariais]
[automatizar processos da empresa]
[automação de tarefas repetitivas]
"automação de processos"
"automatizar processos"
"consultoria em automação"
```

### Grupo 2 — Integração de Sistemas
```
[integração de sistemas empresariais]
[integrar crm com erp]
[integrar sistemas da empresa]
"integração de sistemas"
"conectar sistemas"
"integrar ferramentas"
```

### Grupo 3 — Intenção Alta (bottom of funnel)
```
[empresa de automação de processos]
[contratar automação de processos]
[automação empresarial preço]
[quanto custa automação de processos]
"empresa de automação"
"automatizar minha empresa"
```

## Configurações Críticas

| Configuração | Valor |
|---|---|
| Rede de Display | **DESLIGADO** |
| Parceiros de Pesquisa | **DESLIGADO** |
| Segmentação geográfica | Brasil inteiro ou estado-alvo |
| Lance | CPA desejado ou Maximizar Conversões |
| Conversão | Clique no botão "Agendar" ou agendamento confirmado |
| Orçamento diário mínimo | R$ 30/dia para ter dados suficientes |
| Horário | Seg–Sex, 08h–19h |

## Acompanhamento de Conversão

Configurar conversão no Google Ads como:
1. **Clique no link "Agendar"** na landing page (evento de clique)
2. **Agendamento confirmado** no Google Calendar (se houver página de confirmação)

Sem conversão configurada corretamente, o algoritmo não aprende e o custo por lead sobe.
